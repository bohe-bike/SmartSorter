# SmartSorter 更新日志

## [1.0.0] - 2026-03-23

### 新增
- **规则引擎**：支持 7 种条件字段 × 17 种运算符，AND/OR 嵌套条件组
- **4 种文件操作**：重命名（查找替换/前缀/后缀/序列号）、移动、复制、删除
- **移动/复制**：基路径 + 子目录名双输入框，支持文件夹选择器
- **12 个魔法变量**：`{filename}`, `{extension}`, `{created_year}`, `{seq}` 等模板变量
- **实时预览**：分析预览结果，逐文件显示源→目标 Diff
- **安全执行**：SHA-256 校验的安全移动/复制，执行结果 Toast 通知
- **撤销系统**：支持对移动/重命名/复制操作执行撤销
- **历史日志**：完整的执行记录，含时间、耗时、成功/失败/跳过统计
- **重复文件检测**：大小预筛 + SHA-256 精确去重，支持保留最新/最旧
- **进度条**：重复扫描、预览分析、执行操作均有实时进度显示
- **异步执行**：所有重操作后台线程执行，不阻塞主界面
- **设置页面**：主题切换（系统/亮/暗）、冲突策略、删除确认
- **路径安全**：自动清理路径空白和非法字符，自动创建目标目录
- **左右分栏布局**：左侧数据源+规则配置，右侧预览结果
